• generic ML frameworks and tools
    • TensorFlow
    • PyTorch
    • XLA (Accelerated Linear Algebra): compiler for TensorFlow code; developed by Google.
      • domain-specific compiler for linear algebra that can accelerate TensorFlow models with potentially no source code changes.
    • JAX: kind of a replacement for numpy, developed by Google.
      • The cheapest way to use JAX is to add import jax.numpy as np on the top of the file.
  • RL-specific frameworks
    • Stable Baselines 3: built on PyTorch
      • DLR-RM/stable-baselines3: PyTorch version of Stable Baselines, reliable implementations of reinforcement learning algorithms.
    • Facebook ReAgent: built on PyTorch.
      • facebookresearch/ReAgent: A platform for Reasoning systems (Reinforcement Learning, Contextual Bandits, etc.)
    • Google Dopamine: built on JAX; compatible with tensorflow (legacy).
      • google/dopamine: Dopamine is a research framework for fast prototyping of reinforcement learning algorithms.
  • RL algorithm codebase
    • labml.ai Annotated PyTorch Paper Implementations - Reinforcement Learning Algorithms
  • RL environments
    • OpenAI gym
    • gridworld
      • Farama-Foundation/MiniGrid: Simple and easily configurable grid world environments for reinforcement learning
    • crafter: a cheap version of minecraft. It is an OpenAI gym environment.
      • danijar/crafter: Benchmarking the Spectrum of Agent Capabilities
